## 1. 项目概述

### 1.1 项目名称
Image Background Remover（图片背景移除工具）

### 1.2 项目类型
在线图片处理工具（Web 应用）

### 1.3 核心价值
用户通过简单的拖拽上传操作，即可获得移除背景的透明 PNG 图片，全程无需注册、操作便捷。

### 1.4 目标用户
- 需要快速处理证件照、电商图片的用户
- 需要移除图片背景但不想使用复杂 Photoshop 的用户
- 个人开发者或小型企业

## 2. 功能需求

### 2.1 核心功能（MVP）

| 功能模块 | 功能描述 | 优先级 |
|---------|---------|-------|
| 图片上传 | 支持拖拽或点击上传，支持 JPG/PNG/WebP | P0 |
| AI 去背景 | 调用 RemoveBG API 移除图片背景 | P0 |
| 结果预览 | 展示原图与处理后图片的对比（滑块或并排） | P0 |
| 图片下载 | 下载透明背景 PNG 格式 | P0 |
| 加载状态 | 上传/处理/完成各阶段状态反馈 | P0 |

### 2.2 用户流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐

│             │    │  (拖拽/点击)│    │  (等待结果) │    │  (对比+下载)│
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

## 3. 技术方案

### 3.1 技术栈

| 层级 | 技术选型 | 说明 |
|-----|---------|-----|
| 前端框架 | Next.js 14 (App Router) | TypeScript |
| 样式 | Tailwind CSS | 快速开发、响应式 |
| 上传组件 | react-dropzone | 拖拽上传 |
| 预览 | Canvas API | 原图 vs 结果对比 |
| 后端 | Next.js API Routes | 处理图片转发 |
| AI 服务 | RemoveBG API | 图片背景移除 |
| 部署 | Cloudflare Pages/Vercel | 静态托管 + API Functions |

### 3.2 API 设计

**POST `/api/remove-bg`**

请求：
```
Content-Type: multipart/form-data
Body: { image: File }
```

响应（成功）：
```
Content-Type: application/json
{
  "success": true,
  "data": {
    "result": "data:image/png;base64,...",  // Base64 编码的结果图
    "credits_used": 1
  }
}
```

响应（失败）：
```
{
  "success": false,
  "error": {
    "code": "API_ERROR",
    "message": "RemoveBG API 调用失败"
  }
}
```

### 3.3 数据流

```
用户上传图片 (Base64/Blob)
    ↓
前端 -> POST /api/remove-bg
    ↓
后端接收图片 Buffer
    ↓
后端 -> RemoveBG API (POST with API Key)
    ↓
RemoveBG 返回透明背景图
    ↓
后端 -> 前端 (Base64 JSON)
    ↓
前端 Canvas 预览 + 下载
```

### 3.4 性能考虑

| 场景 | 处理方式 |
|-----|---------|
| 图片大小 | 限制最大 10MB |
| 处理超时 | 前端 30s 超时提示 |
| 大图片 | 建议前端压缩后再上传（可选优化） |
| 内存 | Cloudflare Functions 内存限制 128MB |

## 4. 非功能需求

### 4.1 兼容性
- 浏览器：Chrome、Firefox、Safari、Edge 最新两个版本
- 设备：桌面端优先，移动端可用

### 4.2 安全
- API Key 存储在环境变量，不暴露在前端
- 图片处理在服务端完成，不在前端留存

### 4.3 可用性
- 单页面完成全流程，无需页面跳转
- 错误信息友好提示
- 加载状态全程可见

## 5. MVP 范围边界

### 5.1 做（In Scope）
- ✅ 单张图片上传
- ✅ AI 背景移除
- ✅ 结果预览
- ✅ PNG 下载

### 5.2 不做（Out of Scope）
- ❌ 用户系统/登录
- ❌ 批量处理
- ❌ 背景替换/换色
- ❌ 历史记录
- ❌ 证件照换底色
- ❌ 图片编辑其他功能

> MVP 版本发布后，根据用户反馈再规划下一期功能。

# Image Background Remover - MVP 需求文档

## 📋 产品概述

**产品名称：** Image Background Remover（图片背景移除工具）

**产品定位：** 轻量级在线图片背景移除工具，无需注册，即传即用

**目标用户：**
- 电商卖家（商品图抠图）
- 自媒体创作者（封面图制作）
- 普通用户（证件照、生活照处理）

**核心价值：**
- ✅ 3 秒完成抠图
- ✅ 无需注册，保护隐私
- ✅ 高清 PNG 下载

---

## 🎯 MVP 功能范围

### 核心功能（必须有）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **图片上传** | 支持拖拽上传 + 点击选择 | P0 |
| **格式支持** | JPG, PNG, WebP（最大 5MB） | P0 |
| **背景移除** | 调用 Remove.bg API 处理 | P0 |
| **结果预览** | 展示处理后的透明背景图 | P0 |
| **图片下载** | 下载 PNG 格式（透明背景） | P0 |
| **错误处理** | API 失败/格式错误友好提示 | P0 |

### 增强功能（可选）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **对比视图** | 原图/处理后对比滑块 | P1 |
| **批量处理** | 一次上传多张图片 | P1 |
| **处理历史** | 本地缓存最近处理记录 | P2 |
| **图片编辑** | 裁剪、旋转、调整大小 | P2 |

## 🏗️ 技术架构

### 技术栈

```
┌─────────────────────────────────────────────────────────┐
│                      Cloudflare Pages                    │
│  ┌─────────────┐     ┌──────────────┐     ┌──────────┐ │
│  │  Next.js 14 │────▶│  API Route   │────▶│ Remove.bg│ │
│  │  + TS + CSS │     │  (Edge Func) │     │   API    │ │
│  └─────────────┘     └──────────────┘     └──────────┘ │
└─────────────────────────────────────────────────────────┘
```

| 层级 | 技术 | 说明 |
|------|------|------|
| **前端框架** | Next.js 14 | App Router + Server Components |
| **样式** | Tailwind CSS | 快速构建响应式 UI |
| **部署** | Cloudflare Pages | 免费、全球 CDN、Edge Functions |
| **API** | Remove.bg | 专业背景移除服务 |
| **存储** | 无（内存处理） | 图片不落盘，保护隐私 |

### 数据流

```
用户上传图片 
    ↓
前端压缩/校验（< 5MB）
    ↓
POST /api/remove（FormData）
    ↓
Edge Function 调用 Remove.bg API
    ↓
返回处理后的 PNG（Base64/Blob）
    ↓
前端预览 + 下载
```

## 🔌 API 设计

### 前端 → 后端

**接口：** `POST /api/remove`

**请求：**
```typescript
FormData {
  image: File  // 图片文件
}
```

**响应（成功）：**
```typescript
{
  success: true,
  data: {
    imageBase64: string,  // 处理后的 PNG Base64
    originalSize: number, // 原图大小（字节）
    processedSize: number,// 处理后大小
    processingTime: number// 处理耗时（ms）
  }
}
```

**响应（失败）：**
```typescript
{
  success: false,
  error: {
    code: 'INVALID_FORMAT' | 'FILE_TOO_LARGE' | 'API_ERROR',
    message: string
  }
}
```

### 后端 → Remove.bg

**接口：** `POST https://api.remove.bg/v1.0/removebg`

**Headers：**
```
X-Api-Key: ${REMOVE_BG_API_KEY}
```

**Body：**
```
FormData {
  image_file: File,
  size: 'auto'
}
```

## ⚠️ 错误处理

| 错误类型 | 前端提示 | 处理方式 |
|----------|----------|----------|
| 文件格式不支持 | "不支持的图片格式，请使用 JPG/PNG/WebP" | 前端校验 |
| 文件超过 5MB | "图片太大啦，请上传 5MB 以内的图片" | 前端校验 |
| API Key 无效 | "服务暂时不可用，请稍后重试" | 后端日志 |
| API 额度耗尽 | "今日免费次数已用完，明日再来吧" | 返回友好提示 |
| 网络错误 | "网络连接失败，请检查网络" | 重试机制 |

## 🔐 安全考虑

1. **API Key 保护**
   - Key 存储在 Cloudflare 环境变量
   - 前端无法访问，仅 Edge Function 可调用

2. **图片安全**
   - 图片不落盘，纯内存处理
   - 不记录用户上传图片
   - 不保存处理结果

3. **速率限制**
   - 单 IP 每分钟最多 10 次请求
   - 防止滥用和 DDoS

4. **文件校验**
   - 前端校验文件类型（MIME Type）
   - 后端二次校验文件头

compatibility_date = "2024-01-01"
compatibility_flags = ["nodejs_compat"]

REMOVE_BG_API_KEY = "your-api-key"

```bash
# 开发环境
npm run dev

# 生产部署
npm run build
wrangler pages deploy .next/static --project-name=image-bg-remover
```
